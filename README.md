# Mobilna aplikacija - Slastičarna

## Opis
Ovo je mobilna aplikacija napravljena u MIT App Inventor-u tokom nastave, za online poručivanje torti i kolača.  

⚠️ Projekat je u ranoj fazi i trenutno nisu sve funkcionalnosti implementirane. Trenutno rade samo osnovni buttoni.  

Ipak, aplikacija pokazuje moj rad na:  
- dizajnu ekrana  
- osnovnim interakcijama  
- upoznavanju App Inventor platforme

## Kako otvoriti
1. Idi na [MIT App Inventor](https://appinventor.mit.edu/)  
2. Import → odaberi fajl `aplikacija.aia`
